Mini Judge
A lightweight online code judge built with Java Spring Boot and Vanilla JavaScript.
What it does

Write Java code directly in the browser
Provide custom input before running
Get real-time output instantly
Navigate between multiple coding questions

Tech Stack

Frontend — HTML, CSS, Vanilla JavaScript
Backend — Java, Spring Boot
Compiler — Java Runtime via ProcessBuilder
Testing - POSTMAN(For API testing)

How it works

User writes Java code in the code editor
User provides input in the input box
Frontend sends code + input to Spring Boot backend via REST API
Backend compiles the code using javac and runs it using java
Output is sent back and displayed on screen

Features

Real-time code compilation and execution
Compilation error display
Runtime error display
Multiple questions with Prev/Next navigation
Input/Output based testing like Codeforces and HackerRank
5 second time limit to prevent infinite loops

How to run locally

Clone the repository
Open in Spring Tools Suite or Eclipse
Run as Spring Boot Application
Open QuestionPage.html directly in your browser
Start coding!
